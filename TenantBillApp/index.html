How to use

  1. Type tenant name. If a previous record exists, previous reading and last bill amount will auto-fill.
  2. Enter current reading and rate per kWh, then press Calculate to preview charges.
  3. Add any additional charges (water, internet, etc.) and Save record to keep it in history.
  4. Use Export CSV to download all saved records. Data is stored locally in your browser.
